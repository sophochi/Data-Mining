# Data-Mining

A collection of Jupyter notebooks focused on implementing core data mining and machine learning techniques.

## Project Structure

* **`AssociationRuleLearning.ipynb`** – Discovery of interesting relations between variables using association rule algorithms.
* **`Classification.ipynb`** – Supervised learning models implemented to classify structured data.
* **`Clustering.ipynb`** – Unsupervised learning techniques used to group data points into distinct clusters.
* **`DimensionalityReduction.ipynb`** – Methods for reducing the number of random variables under consideration (e.g., PCA).
